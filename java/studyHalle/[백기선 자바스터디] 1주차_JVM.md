---

* Java Virtual Machine
* Java는 운영체제에 상관 없이 특정 가상 머신 위에서 동작하는 언어로 이때 사용되는  가상 머신을 JVM이라 한다.

#2. 컴파일 하는 방법

---
* $ javac [File Name].java

#3. 실행하는 방법

---
